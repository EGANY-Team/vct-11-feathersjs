## Yêu cầu

- [git][git]

## Cài đặt

```bash
# Clone repo
git clone https://github.com/EGANY-Team/vct-11-feathersjs
cd vct-11-feathersjs

# Install package
npm install

# Development: http://localhost:3030
npm run dev

# Production
npm start
```
